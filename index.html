Simple React Time Picker

Source

Example

Installation

$ npm install --save react-dropdown-timepicker

Usage


import TimePicker from 'react-dropdown-timepicker';

render() {
	<TimePicker
    time={this.state.time}
    onChange={this.handleTimeChange.bind(this)} />
}

Attributes

time

Initially selected time to be highlighted in the time selection grid. May be a JavaScript Date object or a simple object with "hour" and "minute" fields. The hour value should be in the range [0, 23].

onChange

Function to be called when a time is selected. The first argument will be a simple object with "hour" and "minute" fields. The hour value will be 24-hour format in the range [0,23].
